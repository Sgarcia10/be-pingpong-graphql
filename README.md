## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# watch mode
$ npm run start:dev
```

Open [http://localhost:3000/graphql] with your browser to see the graphql playground.

## Deploying the app

```bash
# development
$ npm run build

# serverless deploy
$ sls deploy
```

## Test

```bash
# unit tests
$ npm run test
```
